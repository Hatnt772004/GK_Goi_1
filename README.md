# GK_Goi_1

## Overview
This project includes Jupyter notebooks and Python scripts for financial analysis, data exploration, and machine learning.

## Project Structure
- `notebooks/`: Contains Jupyter notebooks for CAPM analysis, data exploration, and machine learning.
- `src/`: Contains Python scripts for exploratory data analysis.

## Installation
To install the required packages, run:
```
pip install -r requirements.txt
```

## Usage
- Open the Jupyter notebooks in the `notebooks/` directory to explore the analyses.
- Use the `EDA.py` script in the `src/` directory for exploratory data analysis.
